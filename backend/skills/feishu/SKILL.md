# 飞书技能

## 技能描述
连接飞书自定义机器人，通过长连接模式接收和发送消息。

## 快速开始

只需使用 send_feishu_message 工具发送消息，系统会自动处理连接和认证：

```python
# 直接发送消息到你的飞书账号
send_feishu_message(open_id="ou_039c48388eec00cd1b2c157966e5678d", content="你好！这是来自源灵AI的消息")
```

## 工具说明

### send_feishu_message
发送消息到指定用户
- 参数:
  - `open_id`: 接收者的open_id
  - `content`: 要发送的消息内容
  - `msg_type`: 消息类型，默认为"text"

## 注意事项

- 系统已自动配置好你的open_id: `ou_039c48388eec00cd1b2c157966e5678d`
- 无需手动启动长连接，系统会自动管理连接状态
- 无需关心底层实现细节，直接使用工具即可
